Experience

Pelican is a nice library for generating websites where you can use reST to write html pages. Primarily, I choose to use pelican because I can use emacs to write reST pages and rst is painless to write compared to plain old html. In this post, write my experience of getting this site up and running with pelican. Hopefully, it will help someone like me to get started on using pelican. Since this is my first few iterations with pelican settings file looks primitive. Please feel free to send along suggestions for improvements.

Install

Once pip installed straight forward to install ‘pelican’.

$ pip install pelican

Settings file

# -*- coding: utf-8 -*-
AUTHOR = 'Dinesh Weerapurage'
AUTHOR_SAVE_AS = False
SITENAME = 'Overheards, thoughts and tech'
SITEURL = 'http://weerapurage.org'
TIMEZONE = 'US/Eastern'

GITHUB_URL = 'https://github.com/xydinesh/xydinesh.github.com'

DISQUS_SITENAME = "weerapurageorg"
PDF_GENERATOR = False
REVERSE_CATEGORY_ORDER = True
LOCALE = "C"
DEFAULT_PAGINATION = 4
FEED_ALL_RSS = 'feeds/all.rss.xml'
CATEGORY_FEED_RSS = 'feeds/%s.rss.xml'

SOCIAL = (('twitter', 'http://twitter.com/xydinesh'),
      ('github', 'http://github.com/xydinesh'),)

# global metadata to all the contents
DEFAULT_METADATA = (('yeah', 'it is'),)
# Default theme
THEME = 'notmyidea'

Themes

There are good number of pelicen themes available on github where you can fork/clone them and use. Just need to modify settings file (which I called settings.py)

THEME = '../pelican-themes/notmyidea-cms'
