Install ‘Fabric’ on Windows Tweet

Date Fri 07 December 2012 By Dinesh Weerapurage Category Tech. Tags python fabric windows pip pycrypto

Easy

If it works, pretty easy to do with pip

$pip install fabric

Trouble

Trouble started when I got an error while installing fabric on windows. Error seems to come from building pycrypto library.

running build_ext
building 'Crypto.Random.OSRNG.winrandom' extension
warning: GMP or MPIR library not found; Not building Crypto.PublicKey._fastmath.

error: Unable to find vcvarsall.bat

After scrolling through few webpages found a solution on StackOverFlow where it suggests to do following

>SET VS90COMNTOOLS=%VS100COMNTOOLS%

or

>SET VS90COMNTOOLS=%VS110COMNTOOLS%

Depending on your VS installation, and then

$pip install fabric

It gives following error, Argh !

build\temp.win32-2.7\Release\src\winrandom.pyd.manifest : general error c1010070
: Failed to load and parse the manifest. The system cannot find the file specifi
ed.
warning: GMP or MPIR library not found; Not building Crypto.PublicKey._fastmath.

error: command 'mt.exe' failed with exit status 31

Again after scrolling few pages, found an answer on fuyun ‘s website.

For the sake of completeness, I have quoted his/her solution.

Now try to build it again. I got this error:

buildtemp.win32-2.6Release_mysql.pyd.manifest : general error
c1010070: Failed to load and parse the manifest.
The system cannot find  the file specified.

error: command ‘mt.exe’ failed with exit status 31

To fix this problem, go to C:\Python2.6\Lib\distutils,
edit msvc9compiler.py, search for ‘MANIFESTFILE’, you will find the following line

ld_args.append(‘/MANIFESTFILE:’ + temp_manifest)

Then append the following line after the above line,

ld_args.append(‘/MANIFEST’)

then

$pip install fabric
