Experience

Even though I initally planned to use Django with bootstrap , I was impressed after seeing few websites generated using pelican. In this post I write my experience of getting this site up and running with pelican.

Install

Once pip installed straight forward to install 'pelican'. Also I would encourage to install 'fabric' too.

$ pip install pelican

$ pip install fabric

Note

You may run into problem installing fabric on Windows sometimes. I will write another post explaining solutions I used.
